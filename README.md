# 台股股東會紀念品攻略與評估工具 🎁

這是一個專為台灣股民（特別是**零股股東**）打造的股東會紀念品攻略庫。內含一套能自動抓取最新股價並計算「零股性價比 (CP值)」的 Python 評估腳本，以及一個精心設計的互動式網頁前端。

🌐 **線上體驗**：[https://xx8897.github.io/tw-stock-gift-evaluator/](https://xx8897.github.io/tw-stock-gift-evaluator/)

---

## 🧶 策略理念：零股薅羊毛，把小錢放大

> **花幾百元買進 1 股零股，每年參加股東常會，嚕一點紀念品，同時還能收一點股息。**

台灣有許多股票，**單股股價在幾元到幾十元之間**，持有 1 股零股的門檻極低。只要在公司股東會紀念品的「基準日」（通常是除息/除權前）持有至少 1 股，就有資格以零股股東身分請人代領股東會紀念品。

### 為什麼值得做？
- 💸 **成本極低**：許多高 CP 值標的股價不到百元，進場完全無壓力。
- 🎁 **實質回饋**：從超商商品卡、日常用品、到六福村遊樂園門票（市值 1,199 元），有些標的光靠紀念品就能「回本」不只一次。
- 🔁 **年年可領**：只要公司持續發放，每年都可以重複嚕一次，最理想的標的是「五年連續發放」的穩定大戶。
- 🧮 **科學化精算**：本工具透過獨家公式 `（五年紀念品總估值 ÷ 股價）× （發放次數 ÷ 5）`，一眼就能看出哪支是真正超值的神股。

> **⚠️ 免責聲明**：本工具純粹以「紀念品 CP 值」作為排序依據，不構成任何投資建議。任何股票的基本面、業績、流動性等因素請自行評估。

## ✨ 核心功能與特色 (v3 完全體)

### ☁️ 無縫的雲端跨裝置體驗
- **一鍵登入同步**：整合 Google OAuth 2.0，免註冊直接綁定。
- **買入清單帶著走**：手機上標記的「已買入」股票，回到電腦上原封不動在那裡。
- **智能背景儲存**：結合防抖動 (Debounce) 機制，閒置時自動背景連線 Supabase 資料庫，並具備企業級 RLS 安全性隔離。

### 🔍 現代化與高自訂性 UI
- **深色毛玻璃美學**：採用質感極佳的深色主題、流動背景與霓虹元素。
- **多維度精準篩選**：支援 1-5 星級、連續五年發放、已買/未買過濾。
- **(New!) 隱藏身分證領取**：一鍵過濾掉「必須出示身分證正本」的麻煩標的，專心嚕電子投票羊毛！
- **五年歷史資料庫**：881 筆精算資料，每筆紀錄皆可彈窗查看完整五年發放歷史。

### 🤖 全自動化無人值守更新
- **GitHub Actions 排程**：旺季每週、淡季每月自動觸發擷取 TWSE / TPEX 最新收盤價並重算 CP 值，自動部署上線，保證資料永遠最新！

## 🚀 關於 Python 估值腳本 `evaluate_stocks.py`

此腳本是本專案的資料心臟，透過演算法與關鍵字模型推估紀念品價值。

### 🏷️ 紀念品估值模型 v2（參考市場實際行情）
模型經過實際市場調查與二手轉售價格校正（以下為節錄，完整對照請見程式碼）：

| 類型 | 估值 | 依據 |
|---|---:|---|
| **大魯閣** 消費抵用券 | 800 元 | 零股股東獲 800元抵用券 + 8張買1送1，二手轉售約 800~1000 元 |
| **六福村** 主題樂園門票 | 1,199 元 | 2024 年成人門票市價 |
| **王品** 股東大禮包 | 400 元 | 面額 3,000 元但有消費門檻，二手約 400 元 |
| 現金/商品卡 | 面額 | 正規表示式精準抓取 |
| 食用油/不鏽鋼餐具/保溫杯 | 120-150 元 | 市售行情保守估計 |
| ... | ... | ... |

*(腳本執行方式與相依套件：請提前安裝 `pandas`, `requests`, `openpyxl`)*
