# 台股股東會紀念品攻略與評估工具 🎁

這是一個專為台灣股民（特別是**零股股東**）打造的股東會紀念品攻略庫。內含一套能自動抓取最新股價並計算「零股性價比 (CP值)」的 Python 評估腳本。

---

## 📂 目錄檔案說明

- **`2021-2025.xlsx`**：原始的全台股股東會紀念品歷史發放紀錄表（881 筆），資料來源為**零股寶**。
- **`evaluate_stocks.py`**：核心評分腳本，讀取原始資料、抓取最新股價、計算 CP 值與推薦評分。
- **`2021-2025_推薦評分.xlsx`**：腳本產出的**完整評分名單**（881 筆）。
- **`2021-2025_年年發放.xlsx`**：腳本產出的**篩選版名單**，僅保留五年內發放次數 >= 5 的穩定標的。
- **`README.md`**：本說明文件。

---

## 📖 資料來源

原始資料 `2021-2025.xlsx` 取自 **[零股寶](https://www.stockgift.tw/)** — 台灣最大的零股股東會紀念品代領平台。該資料涵蓋 2021 至 2025 年間，全台上市/上櫃公司的股東會紀念品發放紀錄，包含股號、公司名稱、紀念品品項、領取條件、五年內發放歷史等資訊。

---

## 🚀 關於 `evaluate_stocks.py` 評分腳本

讀取原始資料，即時抓取最新收盤價，透過**演算法與關鍵字模型**自動推估紀念品價值，產出兩份推薦名單。

### 📡 股價來源（三層備援）
1. **台灣證交所 (TWSE) API**：抓取所有上市股票的最新收盤價。
2. **櫃買中心 (TPEX) API**：抓取所有上櫃股票的最新收盤價。
3. **原始 Excel「股價」欄位**：若以上兩者都抓不到（例如已下市），則使用零股寶記錄的舊股價。

### ⚙️ 評分邏輯

腳本包含**兩套性價比計算方式**：

#### 舊版（以上次紀念品計算）
> **`性價比 (CP值) = (紀念品預估價值 ÷ 最新股價) × (五年內發放次數 ÷ 5)`**

#### 新版（以五年紀念品總值計算）
> **`新版性價比 = (五年紀念品總估值 ÷ 最新股價) × (五年內發放次數 ÷ 5)`**

- 將「五年發放紀念品」欄位中每一年的品項逐一拆開
- **自動去除年份前綴**（如 `(2025)`）後再送入估值模型
- **跳過「不發放」「未發放」的年份**，不計入總估值

### ⭐ 推薦評級對照
| 評級 | 條件 |
|---|---|
| 5 星 | CP 值 >= 2.0 |
| 4 星 | CP 值 >= 1.0 |
| 3 星 | CP 值 >= 0.5 |
| 2 星 | CP 值 >= 0.1 |
| 1 星 | CP 值 < 0.1 或無發放 |

## 🛠️ 環境與使用方式

### 軟體需求
- **Python 3.x**
- **套件**：`pandas`, `requests`, `openpyxl`

### 執行腳本
```bash
pip install pandas requests openpyxl
python evaluate_stocks.py
```
執行完畢後即會同時生成上述兩份 Excel 檔案。可透過 `新版性價比` 或 `新版推薦評分` 欄位排序，快速找出超值的零股神股！
