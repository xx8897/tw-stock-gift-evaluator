# Identity 系統實作驗證清單

本文件用於驗證 **台股股東會紀念品指南** Identity 系統改版的所有功能與體驗細節。

---

## 🎨 1. 視覺與基礎介面驗證 (Visual & UI)
- [ ] **深色玻璃彈窗 (Dark Glass Modal)**: 彈窗背景具有 `backdrop-filter: blur` 效果，且外觀符合深色質感設計。
- [ ] **頁尾 Email 更新**: 確認 `index.html` 頁尾顯示 `Made with ❤️ by xx8897 (您的信箱)` 且格式正確。
- [ ] **角色適應性按鈕**: 未填寫完成時，送出按鈕為灰色半透明；填寫正確後，按鈕色彩亮起。
- [ ] **暱稱計數器**: 暱稱輸入框右下角顯示 `0/20`，且超過 20 字時無法再輸入。
- [ ] **密碼強度提示**: 輸入密碼時，下方出現對應的強度線條（紅/黃/綠）。

## ⚡ 2. 互動與體驗細節 (UX Polish)
- [ ] **3 秒重量級延遲**: 點擊登入/註冊後，Spinner 持續約 3 秒，且下方動態切換三階段文案。
- [ ] **成功打勾動畫**: 3 秒結束後，按鈕變為綠底 `✓` 符號並維持 0.5 秒後才關閉彈窗。
- [ ] **成功數位彩帶**: 登入/註冊成功瞬間，彈窗背景有輕微的彩帶/紙屑噴發效果。
- [ ] **錯誤輸入抖動 (Shaking)**: 輸入錯誤密碼或無效 Email 時，表單產生左右抖動效果。
- [ ] **信箱拼錯糾錯**: 輸入 `gamil.com` 等常見錯誤時，出現修正提示。
- [ ] **密碼長按偷看**: 長按密碼框眼睛圖示時顯示密碼，鬆開即刻隱藏。
- [ ] **鍵盤流暢操作**: 
    - [ ] `Enter` 鍵可正確跳轉焦點或送出表單。
    - [ ] 彈窗打開時，焦點自動落在 Email 欄位。
- [ ] **漸進式載入**: 彈窗元件由下而上帶著微小時間差滑入。
- [ ] **禮物盒懸停提示**: 滑鼠移至禮物盒顯示「點我有驚喜」。
- [ ] **禮物盒點擊感**: 按下禮物盒時有即時的縮放/擠壓反饋，放開即連動噴發效果。
- [ ] **雲端同步提示**: 滑鼠移至同步按鈕顯示功能說明。
- [ ] **骨架屏載入**: 延遲期間顯示流光效果的骨架預覽。
- [ ] **行動端震動**: 手機操作錯誤或成功時有短促震動。
- [ ] **慶祝來源定位**: 彩帶/金幣噴發位置需對應點擊的按鈕或禮物盒。

## 🔐 3. 功能與流程驗證 (Functional Flow)
- [ ] **註冊流程**: 輸入 Email/密碼 -> 送出 -> 顯示 OTP 表單 -> 收信 -> 輸入 6 碼 -> 成功登入。
- [ ] **OTP 一條龍自動化**: 輸入完第 6 位數字時，自動觸發驗證，不需手動點按鈕。
- [ ] **重新發送 (Cooldown)**: 點擊重寄驗證碼後，按鈕變灰並開始 60 秒倒數。
- [ ] **忘記密碼流程**: 輸入 Email -> 收信點擊連結 -> 回到網頁自動彈出「設定新密碼」-> 成功更新。
- [ ] **持久化登入 (Remember Me)**: 勾選後重啟瀏覽器，登入狀態依然維持。
- [ ] **跨分頁同步**: 分頁 A 登出，分頁 B 自動更新為登出狀態。
- [ ] **分頁標題提醒**: 切換到收信分頁時，本網頁標題顯示 `(1) 📩 驗證進行中...`。

## 🚀 4. 數據整合與進階邏輯 (Data & Logic)
- [ ] **數據合併策略 (Merge vs Overwrite)**:
    - [ ] 訪客有資、雲端有資時，登入後彈出衝突處理視窗。
    - [ ] 測試「合併資料」: 本地與雲端資料取聯集。
    - [ ] 測試「覆蓋本地」: 捨棄訪客資料，以雲端為準。
    - [ ] 新帳號自動同步: 雲端空白時，直接將訪客資料寫入雲端。
- [ ] **登入後進度條**: 登入成功後，網頁頂部跑過一條「數據同步中」的精美進度條。
- [ ] **防手滑關閉**: 在註冊表單填寫中關閉分頁，應跳出 `beforeunload` 警告。
- [ ] **全帳號刪除 (GDPR)**: 設定中點選刪除帳號 -> 彈出確認 -> 清空所有 Database 與 Auth 紀錄。

## 🛡️ 5. 安全與邊緣案例 (Security & Edge Cases)
- [ ] **XSS 防護**: 輸入帶有 `<script>` 的內容，確保不會執行。
- [ ] **離線偵測**: 斷網時點擊登入，顯示「請檢查網路連線」而非系統錯誤。
- [ ] **隱私遮罩 (Masking)**: 成功提示中的 Email 顯示為 `xx****@example.com`。
- [ ] **多次嘗試限制 (429)**: 頻繁點擊時顯示友善的「發送頻率過快」提示。
