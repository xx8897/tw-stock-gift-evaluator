# 台股股東會紀念品攻略與評估工具 🎁

這是一個專為台灣股民（特別是**零股股東**）打造的股東會紀念品攻略庫。內含一套能自動抓取最新股價並計算「零股性價比 (CP值)」的 Python 評估腳本。

---

## 🧶 策略理念：零股薅羊毛

這個工具背後的核心想法很簡單：

> **花幾百元買進 1 股零股，每年參加股東常會，嚕一點紀念品，同時還能收一點股息。**

台灣有許多股票，**單股股價在幾元到幾十元之間**，持有 1 股零股的門檻極低。只要在公司股東會紀念品的「基準日」（通常是除息/除權前）持有至少 1 股，就有資格以零股股東身分請人代領股東會紀念品。

### 為什麼值得做？

- 💸 **成本低**：許多高 CP 值標的股價不到百元，進場門檻低。
- 🎁 **紀念品有實質價值**：從超商商品卡、日常用品、到六福村遊樂園門票（市值 1,199 元），有些標的光靠紀念品就能「回本」不只一次。
- 📈 **股息是加分**：部分標的還有股息，持有成本低，連股息報酬率也相當可觀。
- 🔁 **年年可領**：只要公司持續發放，每年都可以重複嚕一次，最理想的標的是「五年連續發放」的穩定大戶。
- 🧮 **CP 值精算**：本工具的 CP 值公式 = `（五年紀念品總估值 ÷ 股價）× （發放次數 ÷ 5）`，一眼就能看出哪支是真正超值的神股。

### 適合哪些人？

- 想要累積小確幸、收實物福利的投資人
- 想要以極低成本分散持股種類的散戶
- 喜歡研究薅羊毛策略的理財愛好者

> **⚠️ 免責聲明**：本工具純粹以「紀念品 CP 值」作為排序依據，不構成任何投資建議。任何股票的基本面、業績、流動性等因素請自行評估。

## 📖 資料介紹

本專案資料涵蓋 2021 至 2025 年間，全台上市/上櫃公司的股東會紀念品發放紀錄，包含股號、公司名稱、紀念品品項、領取條件、五年內發放歷史等資訊。透過 `evaluate_stocks.py` 自動結合最新股價進行 CP 值評估。

## 🚀 關於 `evaluate_stocks.py` 評分腳本

讀取原始資料，即時抓取最新收盤價，透過**演算法與關鍵字模型**自動推估紀念品價值，產出兩份推薦名單。
> [!NOTE]
> 目前專案內產出的範例名單，其「最新股價」基準日為 **2026/02/25**。

### 📡 股價來源（三層備援）
1. **台灣證交所 (TWSE) API**：抓取所有上市股票的最新收盤價。
2. **櫃買中心 (TPEX) API**：抓取所有上櫃股票的最新收盤價。
3. **原始 Excel「股價」欄位**：若以上兩者都抓不到（例如已下市），則使用零股寶記錄的舊股價。

### ⚙️ 評分邏輯

腳本包含**兩套性價比計算方式**：

#### 舊版（以上次紀念品計算）
> **`性價比 (CP值) = (紀念品預估價值 ÷ 最新股價) × (五年內發放次數 ÷ 5)`**

#### 新版（以五年紀念品總值計算）
> **`新版性價比 = (五年紀念品總估值 ÷ 最新股價) × (五年內發放次數 ÷ 5)`**

- 將「五年發放紀念品」欄位中每一年的品項逐一拆開
- **自動去除年份前綴**（如 `(2025)`）後再送入估值模型
- **跳過「不發放」「未發放」的年份**，不計入總估值

## 🤖 自動化股價更新（GitHub Actions）

本專案透過 **GitHub Actions** 實現全自動化的股價更新流程，無需手動執行腳本：

| 項目 | 說明 |
|---|---|
| **觸發時間** | 每週一 UTC 04:00（台灣時間 中午 12:00） |
| **執行內容** | `evaluate_stocks.py` 抓取 TWSE/TPEX 最新收盤價，重新計算 CP 值，覆寫 `data/` 資料夾中的 xlsx |
| **自動部署** | 資料更新後自動 git push，觸發 GitHub Pages 重新部署 |
| **費用** | 公開 Repo 完全免費 |

> Workflow 設定檔：[`.github/workflows/update-prices.yml`](.github/workflows/update-prices.yml)
> 執行記錄：[Actions 頁面](https://github.com/xx8897/tw-stock-gift-evaluator/actions)

網站標頭的「**最後更新**」時間，即對應 xlsx 最近一次的自動更新日期。
